### Global Components
- **Header**: Logo, navigation menu, donate button

### Main Pages

1. **Home Page (index.html)**
   - Hero section with mission statement
   - About organization
   - Key causes (6 categories)
   - Donation campaign highlights
   - Impact numbers
   - Volunteer recruitment
   - Success stories/testimonials
   - Latest blog articles
   - Newsletter subscription
   - CTA banner

2. **Niche Home Page (index-niche.html)**
   - Issue-focused hero banner
   - Featured emergency campaign
   - Quick donation widget
   - Transparency section
   - Program highlights
   - Urgent causes slider
   - Case studies
   - CTA for donations & volunteering

3. **About Us Page**
   - Organization history
   - Mission, vision & values
   - Leadership team
   - Organization timeline
   - Awards & recognition
   - Partners/donors logos
   - Testimonials
   - Financial transparency

4. **Causes Page**
   - Grid/list layout of all causes
   - Category filters
   - Search functionality
   - Quick CTAs (Donate, Sponsor, Support)

5. **Cause Details Page**
   - Full cause description
   - Donation options (one-time/monthly/custom)
   - Image gallery
   - Impact statistics
   - FAQs
   - Budget breakdown
   - Related causes

6. **Blog Page**
   - Blog grid layout
   - Search bar
   - Category filters
   - Tags filter
   - Pagination

7. **Blog Details Page**
   - Featured image
   - Article content
   - Quote highlight blocks
   - Author information
   - Related articles
   - Sidebar (categories, tags, popular posts)

8. **Contact Page**
   - Contact form
   - Google Map
   - Office address
   - Phone & email
   - Emergency helpline
   - Social media links
